This app is a web mobile first UI for hiring managers to find and hire the best candidates for their company.


Hiring Managers create the jobs and candidates apply for them.

Tinder for jobs. 
The experience should be similar to Tinder for jobs.
The app should be easy to use.
Swipe right to sign up for the job.
Swipe left to decline.

Jobs will have a Card view.
Card view contains: 
Job Photo
Job Title
Job Description
Job Location
Job Compensation
Pills for skills needed for the job
Driver license if required for the job
